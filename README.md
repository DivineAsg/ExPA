# 📚 Exercícios 25–28 | Classes e Sistema de Cadastro e Compra

Este branch reúne os exercícios 25 a 28, com foco em **Programação Orientada a Objetos (POO)** e construção de um **sistema simples de cadastro e compra no console utilizando Python**.

---

## 🧠 Objetivo Geral

Praticar os principais conceitos de programação em Python:

- Classes e objetos
- Métodos
- Variáveis e tipos de dados
- Estruturas de decisão (`if / elif / else`)
- Laços de repetição (`while`)
- Funções
- Listas
- Tratamento de erros (`try / except`)
- Lógica de compras e cálculo de valores

---

## 📌 Exercícios 25, 26 e 27 (POO básica)

### ✔ Exercício 25
Criação da classe `Produto` com os atributos:
- `nome`
- `preco`

### ✔ Exercício 26
Criação de um objeto da classe `Produto` e exibição dos dados.

### ✔ Exercício 27
Implementação de um método dentro da classe para exibir as informações do produto de forma organizada.

---

## 🛒 Exercício 28 – Sistema Integrador

Sistema completo no console com as seguintes funcionalidades:

### 📦 Funcionalidades
- Cadastro de produtos
- Listagem de produtos
- Compra de produtos
- Cálculo do valor total da compra
- Aplicação de regra de desconto
- Tratamento de erros de entrada

---

## 🧱 Estrutura do Sistema

### Classe principal:
- `Produto`
  - `nome`
  - `preco`
  - método de exibição

### Armazenamento:
- Lista de produtos (`list`)
